
Installation
==============

Required Dependencies
-----------------------
 - Drupal:

  - Drupal 8.9.x or 9.0.x (8.9.x recommended)
  - Drupal core modules: Search, Path, View, Entity, and PHP modules.

- PostgreSQL 9.5+
- PHP 7.3+
- UNIX/Linux

Specific Installation Guides
-------------------------------

The recommended method to get involved in Tripal 4 development is Tripal Docker installation. This most closely mimics the enviornment automated tests are run on.
